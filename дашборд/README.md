# Courier Monitor — ВкусВилл

Еженедельный мониторинг рынка труда курьеров в Москве.
Отслеживает условия работы, удержание и стратегии конкурентов.

## Запуск дашборда

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

## Структура

- `hh_collector.py` — вакансии курьеров с HH.ru
- `career_scraper.py` — карьерные страницы конкурентов  
- `claude_search.py` — поиск сигналов через Claude API
- `analyzer.py` — генерация дайджеста
- `run_monitor.py` — мастер-скрипт (запускает всё)
- `dashboard.py` — Streamlit дашборд

## Автозапуск

Через GitHub Actions каждое воскресенье в 10:00 МСК.
Требует секрет `ANTHROPIC_API_KEY` в настройках репозитория.
